# FasoCapitale

Bienvenue sur FasoCapitale — un site web développé avec Termux et publié sur GitHub Pages.

## Objectif
Promouvoir l’innovation numérique au Burkina Faso à travers un projet simple et accessible.

## Technologies utilisées
- HTML
- CSS
- Git & GitHub
- Termux (Android)
